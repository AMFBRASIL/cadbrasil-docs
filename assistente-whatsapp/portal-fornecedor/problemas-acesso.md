# Portal — Problemas de Acesso

## Objetivo

Resolver bloqueios comuns de login e acesso ao portal.

## Quando usar

Não consigo entrar, erro no login, página não abre, conta bloqueada.

## Palavras-chave do cliente

- não consigo entrar
- erro login
- página não abre
- conta bloqueada
- acesso negado
- site fora do ar
- não carrega

## Resposta padrão IA

🇧🇷 **CADBRASIL Oficial ®**

Vamos resolver seu acesso:

**Tente nesta ordem:**
1. Acesse https://fornecedor.cadbrasil.com.br (evite links antigos)
2. Limpe cache do navegador ou use aba anônima
3. Teste outro navegador (Chrome recomendado)
4. Use "Esqueci minha senha" com e-mail cadastrado
5. Verifique se o pagamento foi confirmado (até 24h úteis)

Informe seu **CNPJ** para eu verificar a situação do cadastro.

## PROBLEMA

Não consegue fazer login no portal.

## CAUSA PROVÁVEL

Senha incorreta; pagamento pendente; e-mail diferente do cadastro; cache do navegador.

## QUANDO TRANSFERIR HUMANO

Após orientações sem sucesso; mensagem de conta suspensa; erro técnico persistente 24h+.

## Transferência humana

Cliente seguiu todos os passos 2x sem sucesso; suspeita de bloqueio administrativo.
