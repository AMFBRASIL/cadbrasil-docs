# Portal — Senha / Esqueci Senha

## Objetivo

Ajudar cliente a recuperar ou redefinir senha do portal.

## Quando usar

Esqueceu senha, senha não funciona, quer trocar senha.

## Palavras-chave do cliente

- esqueci senha
- não lembro senha
- redefinir senha
- trocar senha
- senha inválida
- resetar senha

## Resposta padrão IA

🇧🇷 **CADBRASIL Oficial ®**

Para recuperar sua senha:

1. Acesse https://fornecedor.cadbrasil.com.br
2. Clique em **"Esqueci minha senha"**
3. Informe o **e-mail** cadastrado
4. Verifique a caixa de entrada e spam
5. Clique no link recebido e crie nova senha

**Importante:** Nunca enviamos senha por WhatsApp. O link chega só no seu e-mail.

## Exemplos

**Cliente:** "Esqueci minha senha"
**IA:** Passo a passo esqueci senha

## Quando NÃO usar

Cliente nunca fez cadastro — orientar primeiro acesso.

## Transferência humana

E-mail não cadastrado no sistema após verificação de CNPJ; link expirado 3+ vezes.
