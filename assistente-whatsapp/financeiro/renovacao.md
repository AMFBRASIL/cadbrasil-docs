# Financeiro — Renovação (Pagamento)

## Objetivo

Orientar pagamento de renovação anual do serviço CADBRASIL.

## Quando usar

Renovar plano, mensalidade anual, taxa de renovação CADBRASIL.

## Palavras-chave do cliente

- renovar pagamento
- pagar renovação
- mensalidade
- taxa anual
- venceu o plano
- renovar cadastro pago

## Resposta padrão IA

🇧🇷 **CADBRASIL Oficial ®**

Para renovar o serviço CADBRASIL:

1. Informe seu **CNPJ**
2. Envio o link de pagamento
3. Após confirmação (até 24h úteis), continue no **Assistente CADBRASIL**

A renovação do **SICAF no governo** e a renovação do **serviço CADBRASIL** são processos complementares — o Assistente orienta ambos.

## Fluxo IA

CNPJ → solicitar-boleto se cliente pedir link → não negociar valor

## Exemplos

**Cliente:** "Preciso pagar a renovação"
**IA:** Pedir CNPJ → boleto via API

## Quando NÃO usar

Renovação só de certidões SICAF sem pagamento — usar `cliente-renovacao.md`.

## Transferência humana

Pedido de desconto, parcelamento ou contestação de cobrança.
