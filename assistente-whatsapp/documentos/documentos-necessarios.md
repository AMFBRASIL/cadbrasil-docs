# Documentos Necessários

## Objetivo

Orientar quais documentos o cliente precisa para o SICAF.

## Quando usar

Cliente pergunta o que enviar, quais documentos, o que precisa.

## Palavras-chave do cliente

- quais documentos
- o que preciso enviar
- documentação necessária
- que papéis
- preciso mandar o quê

## Resposta padrão IA

🇧🇷 **CADBRASIL Oficial ®**

Os documentos variam conforme o nível do SICAF, mas em geral:

- **Contrato social** / última alteração
- **Cartão CNPJ** atualizado
- **Certidões** (federal, estadual, municipal, FGTS, trabalhista)
- **Certificado digital e-CNPJ** (A1 ou A3)
- Dados dos sócios

O **Assistente CADBRASIL** indica exatamente o que enviar em cada etapa:

🔐 https://fornecedor.cadbrasil.com.br

## Exemplos

**Cliente:** "Quais documentos preciso?"
**IA:** Lista geral + Assistente mostra por etapa

## Quando NÃO usar

Erro ao enviar documento → `problemas/erro-documento.md`. Dúvida sobre certificado → `certificado-digital/`.

## Transferência humana

Documento incomum exigido por edital específico.
